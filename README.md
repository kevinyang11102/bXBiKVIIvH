# 充电桩管理系统 python599

## 项目概述


## 技术栈

### 前端

- HTML
- JavaScript
- CSS
- Bootstrap 5

### 后端

- Flask

### 数据库

- SQLite

## 功能模块

### 用户功能

- 浏览电站信息（位置）
- 浏览充电信息（是否空闲）
- 预约充电桩
- 管理个人预约信息
  - 开始充电
  - 结束充电
  - 预计充满时间
- 查看个人订单信息
- 支付订单
- 查看个人报修信息
- 查看个人反馈信息

### 管理员功能

- 管理电站信息
- 管理充电桩信息
- 管理预约信息
- 管理订单信息

## 系统角色

系统分为用户角色和管理员角色，用户可以进行充电预约和订单管理，管理员负责电站和充电桩的日常运营管理。

## 运行环境

- Python 3.7 或更高版本
- Flask 2.0 或更高版本

具体的目录结构如下：

```plaintext
|-- ...
```
